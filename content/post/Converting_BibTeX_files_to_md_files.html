This is my first post! YAY! While wondering around the amazing world blog writing with the blogdown package I had to choose a theme to work with. For now, I chose the hugo academic theme due to recommendations over the web and it’s so called easy interface and “out of the box” utilities.

The first problem that I had to tackle was how to post my publications and thus to convert my library of publications which were saved as one BibTeX file to multiple .md files and multiple .bib files. Luckily I found a package that was written just for that.

I had to start with export the citation of my publication to a BibTeX file.

Next, I had to install from github the required package.

# If not installed, you should install "devtools" and "bib2academic"
if (!require("devtools")) install.packages("devtools")
## Loading required package: devtools
if (!require("bib2academic")) devtools::install_github("petzi53/bib2academic")
## Loading required package: bib2academic
#Load package
require(bib2academic)

If you’d like to try while reading, I saved my BibTeX file under the folder of the post.

So, now we need to start the conversion using the “bib2acad” function. Note that instead of writing a direct path, I “pasted” the working directory and the directory of the file.

bib2acad(paste(getwd(),
               "content/post/Converting_BibTeX_files_to_md_files/my_articles.bib", 
               sep = "/"),
         copybib = TRUE, abstract = TRUE, overwrite = TRUE)
## Cannot find file 'C:/Amir/Dropbox/MySite/AmirDJV.github.io/content/post/content/post/Converting_BibTeX_files_to_md_files/my_articles.bib'. Check path and/or file name.

If everything went smoothly, we should now see in our working directory two new folders “my-bib-folder” (which holds all the BibTeX files) and “my-md-folder (which holds all the .md files).

Next, we should move these files to there correct path. To do that, we should first get a list of all the files:

bibFiles <- list.files("my-bib-folder", full.names = TRUE)
mdFiles <- list.files("my-md-folder", full.names = TRUE)

Now we can copy them to the correct path:

file.copy(from = bibFiles, to = "static/files/citations/")
## logical(0)
file.copy(from = mdFiles, to = "content/publication/")
## logical(0)

Lastly we should delete the temp folders we created

unlink("my-bib-folder", recursive = T, force = T)
unlink("my-md-folder", recursive = T, force = T)
